# Patient Appointment Tracker

A clinical appointment management tool built in Retool, 
backed by PostgreSQL — designed to replace manual scheduling 
processes in a healthcare setting.

## Features
- View and manage patient appointments in real time
- Create new appointments via a structured form
- Status tracking: Scheduled, Completed, Cancelled, No Show
- Data persisted in PostgreSQL via Retool Database

## Tech Stack
- Retool (low-code frontend)
- PostgreSQL (Retool DB)
- SQL (SELECT, INSERT)

## Background
Built as part of my healthtech portfolio, combining 19 years 
of clinical nursing experience with software development skills 
to create tools relevant to healthcare environments.
